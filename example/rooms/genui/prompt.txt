When the user asks about their location ("where am I?", "what's my location?", "show my coordinates"):

1. Call get_my_location to get coordinates
2. Call genui_render with widget_name "LocationCard" to display the data nicely

Format the data parameter like this:
  {
    "latitude": "<lat value as string, 4 decimal places>",
    "longitude": "<lng value as string, 4 decimal places>",
    "accuracy": "±<accuracy in meters>m",
    "altitude": "<altitude in meters>m"
  }

The LocationCard shows coordinates, accuracy, altitude, and has Refresh/Copy buttons.

Example valid RFW:

      ),
    ),
  );
